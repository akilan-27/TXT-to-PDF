# 📖 About the Project

The TXT to PDF Converter is a lightweight yet powerful web platform developed to quickly transform simple text notes and Markdown files into structured, print-ready PDF documents without requiring heavy software installations.

Traditionally, converting simple text to PDF required bloated word processors or online services that compromise user privacy. This project operates entirely within the user's browser, ensuring speed and confidentiality.

---

# 🚨 Problem Statement

Users often struggle to convert raw text notes into portable and professional documents quickly.

The key challenges included:

* Relying on slow, resource-heavy applications (like MS Word) just to convert a `.txt` file.
* Using online converters that force users to upload their sensitive documents to third-party servers.
* Lack of simple, free tools that support rich text elements (Markdown) without complex setups.
* Poor quality PDFs generated by simple text editors.

# 🔄 Application Workflow

### User Journey

1. User visits the web application.
2. User types text or drops a `.txt` / `.md` file into the editor.
3. User adjusts formatting settings (margins, fonts, paper size).
4. System generates a live preview instantly.
5. User clicks "Export".
6. PDF is generated locally and downloaded directly to the device.

---

# 🔒 Privacy & Security

Security measures implemented:

* **No Backend Server**: The application operates entirely via client-side JavaScript.
* **XSS Protection**: Utilizing `DOMPurify` to sanitize all Markdown inputs before rendering.
* **No File Tracking**: Files are processed in memory and immediately discarded.

# 🧩 Challenges Faced

During development, several challenges were encountered:

* Paginating long documents correctly across multiple PDF pages.
* Maintaining crisp text resolution when converting HTML to Canvas.
* Handling complex Markdown features (like tables and code blocks) smoothly.
* Optimizing performance to ensure the browser doesn't freeze on large `.txt` files.
* Detecting file encodings accurately to prevent garbled text imports.
